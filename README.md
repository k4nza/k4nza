## 🚀 Tentang Saya

Saya seorang pengembang yang sangat antusias dengan **solusi IoT** dan **embedded systems**. Fokus utama saya saat ini adalah menciptakan sistem otomasi cerdas untuk dunia pertanian (Smart Agriculture) melalui proyek **Kanza Smart Garden**.

**Saat ini saya sedang fokus mengerjakan:**
- 🌿 **Smart Hydroponic Systems** - Otomatisasi dosis nutrisi menggunakan ESP32 (V3.1.10).
- 📡 **Integrasi IoT & Sensor** - Konektivitas WiFi dan monitoring data secara real-time.
- 🔧 **Pengembangan Firmware** - Menulis kode C/C++ yang optimal, stabil, dan aman.
